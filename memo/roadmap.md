# 产品路线图

## 阶段规划

### 阶段 1: 独立悬浮球 (当前)
- **目标**: Electron 桌面应用，悬浮球点击录音，松开转文字
- **语音后端**: Python 子进程 (复用现有 stt.py)
- **状态**: 规划中

### 阶段 2: OpenCode 插件
- **目标**: 将悬浮球功能集成为 OpenCode 插件
- **依赖**: 阶段 1 完成并稳定运行
- **状态**: 待启动

---

## 远期规划

### 语音识别后端迁移
- **当前**: Python 子进程 (Whisper/Moonshine/Faster-Whisper)
- **目标**: 纯 Node.js 方案 (whisper.cpp / whisper-node)
- **优势**:
  - 无需 Python 环境
  - 打包更简单
  - 用户体验更好（一键安装）
- **挑战**:
  - whisper.cpp 绑定可能有兼容性问题
  - 需要重新调研和测试
- **优先级**: 中
- **预计时间**: 阶段 2 完成后

---
